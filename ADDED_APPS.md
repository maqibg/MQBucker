## 使用方法

```powershell
# 添加bucket
scoop bucket add mqbucker https://github.com/your-username/MQBucker

# 安装软件
scoop install mqbucker/sparkle
scoop install mqbucker/ani
scoop install mqbucker/tdl
# ... 其他软件
```

1. 部分软件需要管理员权限
2. 某些软件可能需要额外的运行时依赖
3. 建议在安装前检查软件的系统要求
4. Hash值需要手动更新为实际值才能正常使用
